# Search-engine
日本語版wikipediaデータ用検索エンジン

# Usage
`./search`

各種コマンドを入力  
t: SQLテーブルが存在しない場合、作成  
i: wikidataに追加した文章データをSQLに書き込む  
c: SQLに保存されているデータを表示  
d: 単語の辞書用SQLテーブル作成  
g: コマンドiによって追加されたデータから分割単語辞書作成  
l: 単語辞書表示  

s: 検索(4文字以上の言葉を入力して下さい)
